    MapLibre Native

    GitHub Android API Documentation iOS API Documentation

    Open-source SDK for Android and iOS allowing displaying maps inside of your mobile applications, desktop application, or embedded devices. This toolset grants fast maps displaying in iOS and Android apps using the same GPU-based acceleration as the JavaScript version.

    MapLibre-Native is an open source fork of mapbox-native with additional functionality.
