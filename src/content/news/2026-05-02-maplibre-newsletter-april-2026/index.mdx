## 🌐 MapLibre GL JS
Version 5.22.0, 5.23.0 and 5.24.0 were released this month.
These version introduced mostly performance improvements and bug fixes along minor public API additions.
They also mark the last version 5 releases.
We have started releasing version 6 pre-releases with the breaking changes we aim to do as part of this breaking change versions. We will continue to release those in the next few weeks and hopefully finish shortly.
The major breaking changes we aim to do as part of this version are:
- Dropping support for webgl1 in order to be able to fix some long lasting bugs and feature request alongside improving performance and better compatibility with modern libraries.
- Dropping commonjs build and use ESM. This will allow us to move faster, adopt new tools and in general modernize our build system and infrastructure.

Once we have all the breaking change in place we will create a migration guide to help out migrating from version 5 to version 6. Most of the changes we've seen so far are fairly trivial, but might be tricky in some cases, so feel free to try out these pre-releases and help us create good migration documentation for future developers.

We haven't released a breaking change version for over a year now and we believe this is a good time to say goodbye to some old stuff that we have dragged for a long time.
