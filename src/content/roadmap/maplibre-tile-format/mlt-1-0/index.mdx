MLT 1.0 is a modern, MVT-compatible vector tile format.
It offers feature parity with [Mapbox Vector Tile](https://github.com/mapbox/vector-tile-spec), delivering up to 6× better compression on large tiles and faster decoding, while remaining a drop-in successor for existing tile pipelines.

We [announced MLT 1.0](/news/2026-01-23-mlt-release/) in January 2026.
See the [implementation status](https://maplibre.org/maplibre-tile-spec/implementation-status/) page for the current state of integrations.
