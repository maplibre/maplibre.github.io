Martin should be the only service that one needs to run a maplibre map.
As such, we currently, support
- Tiles
- Fonts
- Sprites

Another important part is serving of styles, while technically simple, this enables a variety of new functionality.

GitHub Issues: [#1607](https://github.com/maplibre/martin/issues/1607)
Specification: [MapLibre Style Spec](https://maplibre.org/maplibre-style-spec/)
